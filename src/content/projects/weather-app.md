## Project Overview

A clean, intuitive weather application that provides real-time weather information for any location. Built with React and styled with TailwindCSS for a modern, responsive interface.

## Features

- **Current Weather**: Display temperature, humidity, wind speed, and conditions
- **Location Search**: Search for any city and get instant weather data
- **Geolocation**: Auto-detect user's location on first visit
- **Multiple Units**: Toggle between Celsius and Fahrenheit
- **Hourly Forecast**: See weather predictions for the next 12 hours
- **Responsive Design**: Optimized for mobile, tablet, and desktop

## Technologies

- **React**: Component-based UI
- **TailwindCSS**: Styling and responsive design
- **OpenWeatherMap API**: Real-time weather data
- **Axios**: HTTP client for API requests

## What I Learned

- Working with third-party APIs and handling async data
- State management with React hooks (useState, useEffect)
- Responsive design patterns
- Error handling and user feedback

This project demonstrates full-stack frontend development skills and API integration.
