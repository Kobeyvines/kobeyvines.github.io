## Project Overview

This portfolio website showcases my work, blog posts, and professional experience. It's designed to be fast, visually appealing, and easy to maintain.

## Key Features

- **Content Collections**: Blog posts and projects are managed through Markdown files
- **Dynamic Routing**: Automatic page generation for each blog post and project
- **Responsive Design**: Works seamlessly on all device sizes
- **Dark Mode**: Built-in dark mode toggle for better accessibility
- **Performance**: Optimized images and minimal JavaScript for fast loading

## Technologies Used

- **Astro**: Modern static site generator
- **TailwindCSS**: Utility-first CSS framework
- **TypeScript**: For type safety
- **Markdown**: Content management

## Highlights

- Optimized Core Web Vitals (near-perfect Lighthouse scores)
- Automatic deployment to GitHub Pages
- SEO-friendly structure with meta tags and OpenGraph support
- RSS feed for blog subscribers

This project serves as both a portfolio and a demonstration of modern web development best practices.
