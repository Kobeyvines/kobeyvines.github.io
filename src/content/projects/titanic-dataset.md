## Project Overview

An end-to-end machine learning pipeline that predicts passenger survival on the Titanic using an ensemble learning approach. The project combines SVM and Random Forest classifiers in a Voting Classifier for robust predictions. It features a modern FastAPI backend, containerized with Docker, and includes a live web interface deployed on Render.

## Features

- **Ensemble Learning**: Combines SVM and Random Forest models through a Voting Classifier for improved accuracy
- **FastAPI Backend**: High-performance REST API for predictions
- **Interactive Web Interface**: User-friendly frontend for making survival predictions
- **Docker Containerization**: Fully containerized for consistent deployment across environments
- **Live Deployment**: Accessible web interface hosted on Render
- **Data Preprocessing**: Comprehensive data cleaning and feature engineering
- **Model Evaluation**: Detailed performance metrics and validation results

## Technologies

- **Python**: Core language for data science and backend development
- **Jupyter Notebook**: Exploratory data analysis and model development (96.9% of codebase)
- **Scikit-learn**: Machine learning algorithms (SVM, Random Forest, Voting Classifier)
- **FastAPI**: Modern web framework for building the prediction API
- **Docker**: Containerization for reliable deployment
- **JavaScript/HTML/CSS**: Frontend web interface
- **Render**: Cloud platform for deployment
