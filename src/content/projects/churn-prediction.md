## Project Overview

A comprehensive data science case study investigating and improving a churn prediction model for a mobile money platform. This project demonstrates the complete data science workflow from exploratory data analysis through model rebuilding and validation, with a focus on practical insights and performance optimization.

## Key Components

- **Exploratory Data Analysis (EDA)**: In-depth investigation of customer data patterns, churn distribution, and feature relationships
- **Data Preprocessing**: Cleaning, feature engineering, and preparing data for modeling
- **Model Development**: Building and iterating on churn prediction models
- **Model Validation**: Comprehensive evaluation and performance metrics
- **Insights & Recommendations**: Actionable findings for reducing customer churn

## Features

- **Comprehensive EDA**: Visualizations and statistical analysis of key churn drivers
- **Feature Engineering**: Creation of meaningful predictors from raw data
- **Model Comparison**: Evaluation of multiple algorithms and approaches
- **Performance Metrics**: Detailed analysis using precision, recall, F1-score, and ROC curves
- **Business Insights**: Translation of model results into actionable recommendations

## Technologies

- **Python**: Core data science language
- **Jupyter Notebook**: Interactive analysis and documentation
- **Pandas**: Data manipulation and analysis
- **Scikit-learn**: Machine learning algorithms
- **Matplotlib & Seaborn**: Data visualization
- **NumPy**: Numerical computing

## What I Learned

- End-to-end machine learning workflow and best practices
- Exploratory data analysis techniques and visualization strategies
- Feature engineering and selection methods
- Model evaluation and performance optimization
- Translating technical findings into business insights
- Documentation and reproducible research practices

This project demonstrates proficiency in data science fundamentals, statistical analysis, machine learning, and the ability to derive actionable insights from complex data.
